PACMAN Artificial Intelligence

This was built using Python, allows PACMAN to find food anywhere in a maze. This was done by implementing Breadth First, Depth First, Uniform Cost, and A* search algorithms. I utilized heuristic function to minimize the nodes expanded when search through the maze to find the food

Linear Algebra Suite

This features a suite of linear algebra applications, coded in Python, used to solve common computational linear algebra problems. Includes scripts that can compute matrix multiplication, dot product, cross product, etc.

Basketball Stat Keeping Project

This project was written in C++. The program reads in a .txt file and that data is what is used in the program. You can add games, display individual players or the whole team, delete players, and sort by jersye and name. When the program is closed, it writes another .txt file with the final assorment of players and stats.
