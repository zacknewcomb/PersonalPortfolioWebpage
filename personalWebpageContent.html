Portfolio

This is my current portfolio, and new projects are added and updated regularly! I am constantly working on Python, C++, and web development projects.

Scientific Computation Package

This features a suite of scientific computing applications, coded in Python, used to solve ODEs, solve common linear algebra problems, find roots of real valued functions, and more. It features select algorithms proposed in recent numerical analysis research papers, along with more classic numerical methods.

PACMAN Artificial Intelligence

This was built using Python, allows PACMAN to find food anywhere in a maze. This was done by implementing Breadth First, Depth First, Uniform Cost, and A* search algorithms. I utilized heuristic functions to minimize the nodes expanded when search through the maze to find the food

Space Shooter

A replica of the popular 1980's fixed shooter game, Galaga. Implemented using JavaScript, this top down scrolling shooter game is played in a web browser. It records your score, enemies spawned, enemies defeated, and time you stayed alive.

Basketball Stat Keeping Project

This project was written in C++. The program reads in a .txt file and that data is what is used in the program. You can add games, display individual players or the whole team, delete players, and sort by jersey and name. When the program is closed, it writes another .txt file with the final assorment of players and stats.
